# Anga's Portfolio

## Commands to run this:

First, clone the repository:
```bash
git clone git@github.com:Anga205/Portfolio.git
cd Portfolio
```



Then, install the dependencies:
```bash
bun install
```

then you can run the development server:
```bash
bun run dev
```
